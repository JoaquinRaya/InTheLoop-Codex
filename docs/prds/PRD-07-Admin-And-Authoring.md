# PRD-07: Admin & Authoring

## Goal
Enable fast question lifecycle management without approval bottlenecks.

## Admin capabilities
- Create questions and schedules
- Target whole company, manager subtree, or arbitrary group
- Preview future question resolution by date/profile

## Editing policy
- Editable until first display event
- Immutable after first display

## Out of scope
- Import/export workflows
- Multi-stage approval chains
