# PRD-06: Dashboard & Analytics

## Goal
Provide transparent org-level insights while preserving threshold-based anonymity rules.

## Access model
All users can access dashboards (as specified).

## Filters
- Manager scope: direct reports or recursive full org
- Role
- Level
- Category
- Tags
- Time period

## Engagement chart
- Daily average score across respondents only
- Scores normalized to 1-100

## Drill-down
- Question list with score, count, and previous occurrence delta
- Comment view on question click (verbatim)

## Insufficient data behavior
Display explicit error message when the filter window is below threshold.
